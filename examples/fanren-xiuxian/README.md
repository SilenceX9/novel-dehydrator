# 示例：凡人修仙传（忘语）前 5 章

本目录为 Novel Dehydrator 的效果演示，取《凡人修仙传》前 5 章正文作为测试素材。

- `raw/`：原始章节文本（从 EPUB 提取）
- `dehydrated/`：AI 脱水版本（保留核心剧情 + 结构化元数据）

## 脱水格式说明

每个脱水文件包含两部分：

```
[脱水正文]
...保留核心剧情的精简叙述，约为原文 25%...

---CHAPTER_META---
本章剧情：[一句话剧情概述]
关键人物：[人物名-角色描述；多人用分号分隔]
伏笔线索：[本章埋下的伏笔或悬念]
```
